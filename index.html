Introduction

Sabermetrics is probably the most advanced field in the world of sports analytics. This project aims to address the following question in the field: if I were a batter and I wanted to know how similar the pitches thrown to me by two different pitchers are, how would I do it?

To answer this question, we will analyze pitches thrown to one of the best hitters in the sport Aaron Judge by two of the pitchers who have faced him the most: Kevin Gausman and José Berrios. Apart from the fact that both pitchers have faced Judge more than any other pitcher, another reason to choose them is because both are right-handed and from the same team, the Toronto Blue Jays, which makes the analysis more consistent.

In order to figure this correlation between pitches, we use Lee’s L statistic which is a bivariate spatial correlation coefficient which measures the association between two sets of observations made at the same spatial sites:

\[ L_{x,y} = \frac{N}{\sum_i \left(\sum_j w_{ij}\right)^2} \frac{ \sum_{ij} (\tilde{x}_i - \bar{x})(\tilde{y}_i - \bar{y}) }{ \sqrt{\sum_i (\tilde{x}_i - \bar{x})^2} \sqrt{\sum_i (\tilde{y}_i - \bar{y})^2} } \]

seasons = c(2016, 2017, 2018, 2019, 2020, 2021, 2022, 2023, 2024, 2025)

fetch_season_data = function(season) {
  url = paste0(
    'https://baseballsavant.mlb.com/statcast_search/csv?all=true',
    '&hfGT=R%7C',
    '&hfSea=', season, '%7C',
    '&player_type=batter',
    '&batters_lookup%5B%5D=592450',
    '&min_pitches=0&min_results=0',
    '&group_by=name',
    '&sort_col=pitches',
    '&player_event_sort=h_launch_speed',
    '&sort_order=desc',
    '&min_abs=0',
    '&type=details'
  )
  
  message("Fetching season: ", season)
  
  tryCatch({
    df = read.csv(url)
    df$season = season
    return(df)
  }, error = function(e) {
    warning("Failed to fetch season ", season, ": ", e$message)
    return(NULL)
  })
}

Grid Creation and Correlation Test

In order to calculate Lee’s L statistic, we need to create a grid similar to the one in the plot. We then calculate the proportion of pitches thrown by each pitcher in each cell and use these proportions to compute Lee’s L statistic.

Code 
x_breaks = seq(-3, 2.5, by = 0.5)
z_breaks = seq(-1, 5, by = 0.5)

Matchup = Matchup %>% 
  mutate(
    x_bin = cut(plate_x, breaks = x_breaks, include.lowest = TRUE),
    z_bin = cut(plate_z, breaks = z_breaks, include.lowest = TRUE)
  )

grid_counts = Matchup %>%
  group_by(pitcher, x_bin, z_bin) %>%
  summarise(count = n(), .groups = 'drop')

grid_counts = pivot_wider(
  grid_counts, 
  names_from = pitcher, 
  values_from = count, 
  values_fill = list(count = 0)
)

x_mids = (x_breaks[-length(x_breaks)] + x_breaks[-1]) / 2
z_mids = (z_breaks[-length(z_breaks)] + z_breaks[-1]) / 2


full_grid = expand.grid(x_bin = levels(Matchup$x_bin), 
                        z_bin = levels(Matchup$z_bin))

grid_counts_full = full_grid %>%
  left_join(grid_counts, by = c("x_bin", "z_bin")) %>%
  mutate(
    `592332` = replace_na(`592332`, 0),
    `621244` = replace_na(`621244`, 0),
    x_mid = x_mids[as.numeric(x_bin)],
    z_mid = z_mids[as.numeric(z_bin)],
    prop_gausman = `592332` / sum(`592332`),
    prop_berrios = `621244` / sum(`621244`)
  ) %>%
  arrange(z_bin, x_bin)

With our grid created we can now compute Lee’s L statistic to measure the spatial correlation between the pitch distributions of Kevin Gausman and José Berrios. We use the spdep package to create a spatial weights matrix based on the grid structure, and then we apply the lee.mc function to compute the statistic with a permutation test for significance.

Code 
neighbors_list = cell2nb(nrow = length(z_mids), 
                         ncol = length(x_mids), 
                         type = "queen")
neighbors_list = nb2listw(neighbors_list, style = "W", zero.policy = TRUE)

lee.mc(grid_counts_full$prop_gausman,
       grid_counts_full$prop_berrios,
       neighbors_list,
       nsim = 9999,
       alternative = "greater")

data.frame(
  Parameter = c("Lee's L statistic", "Observed rank", "p-value", "Alternative", "Simulations"),
  Value     = c("0.52458", "10,000 / 10,000", "0.0001", "greater", "9,999 + 1")
) |>
  gt() |>
  tab_header(title = "Lee's L — Monte Carlo results") |>
  tab_style(
    style = cell_fill(color = "#EAF3DE"),
    locations = cells_body(rows = Parameter == "p-value")
  )

The results show a Lee’s L statistic of 0.52458, which indicates a moderate positive spatial correlation between the pitch distributions of Kevin Gausman and José Berrios. The observed rank of 10,000 out of 10,000 in the permutation test suggests that the observed Lee’s L is greater than all the values obtained from the random permutations, leading to a p-value of 0.0001. This indicates that the similarity in pitch distributions between the two pitchers is statistically significant.

From both heatmaps we can observe a tendency of throws directed to the bottom right corner of Aaron Judge’s strike zone, which is a common strategy used by pitchers against right-handed hitters. This area is often targeted because it can be more difficult for the batter to make solid contact, especially if they are trying to hit the ball with the sweet spot of the bat.

Nevertheless, this reasoning can be developed in future work by analyzing the area of the strike zone in which Aaron Judge has the better batting statistics.

Conclusion and further work

This project aims to expand further the Sabermetrics literature by introducing a new method to analyze the similarity of pitches thrown by different pitchers to the same batter. By applying Lee’s L statistic, we can quantify the spatial correlation between the pitch distributions of Kevin Gausman and José Berrios when facing Aaron Judge. The results indicate a significant positive correlation, suggesting that both pitchers have a similar approach when pitching to Judge.

This analysis can be further expanded by including more pitchers and comparing their pitch distributions against Aaron Judge, or even against other batters. Additionally, we could analyze the effectiveness of the pitches in different zones of the strike area by looking at the batting outcomes (e.g., hits, strikeouts) associated with each zone. We can also add an explanation of why pitchers might target specific zones against certain batters based on their strengths and weaknesses, which would provide a more comprehensive understanding of the strategic elements in pitching.

We also encourage the use of spatial data in the field of sports analytics in general, as it can provide valuable insights that are not captured by traditional statistics.

The results show a Lee's L statistic of 0.52458, which indicates a moderate positive spatial correlation between the pitch distributions of Kevin Gausman and José Berrios. The observed rank of 10,000 out of 10,000 in the permutation test suggests that the observed Lee's L is greater than all the values obtained from the random permutations, leading to a p-value of 0.0001. This indicates that the similarity in pitch distributions between the two pitchers is statistically significant.

## Conclusion and further work

This project aims to expand further the Sabermetrics literature by introducing a new method to analyze the similarity of pitches thrown by different pitchers to the same batter. By applying Lee's L statistic, we can quantify the spatial correlation between the pitch distributions of Kevin Gausman and José Berrios when facing Aaron Judge. The results indicate a significant positive correlation, suggesting that both pitchers have a similar approach when pitching to Judge.

This analysis can be further expanded by including more pitchers and comparing their pitch distributions against Aaron Judge, or even against other batters. Additionally, we could analyze the effectiveness of the pitches in different zones of the strike area by looking at the batting outcomes (e.g., hits, strikeouts) associated with each zone. We can also add an explanation of why pitchers might target specific zones against certain batters based on their strengths and weaknesses, which would provide a more comprehensive understanding of the strategic elements in pitching.

We also encourage the use of spatial data in the field of sports analytics in general, as it can provide valuable insights that are not captured by traditional statistics.
